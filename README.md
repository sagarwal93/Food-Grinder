# Food-Grinder
You'll know it when you see it.

## Installation
### Server
`npm install`

### Client
Normal `npm install` and plus a specific version of react-router:
`npm install --save react-router@1.0.0-rc3`
Recommend: `npm install -g webpack webpack-dev-server`

## Running
### Server
`npm run start`

## Client
`webpack-dev-server`
